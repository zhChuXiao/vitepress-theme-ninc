# 按键标记

按键标记用于在文档中展示键盘按键名称，常用于快捷键说明、操作指引与命令行示例。本主题通过双百分号 `%%` 语法将其渲染为带样式的 `<span class="keybutton">` 元素，无需手动编写 HTML。

## 语法

用双百分号 `%%` 包裹按键名，会被转换为带 `keybutton` class 的 `<span>`：

```
%%Ctrl%% + %%C%%
```

渲染后的 HTML：

```html
<span class="keybutton">Ctrl</span> + <span class="keybutton">C</span>
```

## 渲染效果

按下 %%Ctrl%% + %%C%% 即可复制选中内容，%%Ctrl%% + %%V%% 粘贴，%%Enter%% 提交表单。

其他常见写法：

- 单键：%%Enter%%
- 组合键：%%Ctrl%% + %%Shift%% + %%N%%
- 功能键：%%F5%%

## 源码示例

````md
按下 %%Ctrl%% + %%C%% 即可复制选中内容，%%Ctrl%% + %%V%% 粘贴，%%Enter%% 提交表单。

其他常见写法：

- 单键：%%Enter%%
- 组合键：%%Ctrl%% + %%Shift%% + %%N%%
- 功能键：%%F5%%
````

## 常见按键组合

- **复制 / 粘贴**：%%Ctrl%% + %%C%% / %%Ctrl%% + %%V%%
- **撤销 / 重做**：%%Ctrl%% + %%Z%% / %%Ctrl%% + %%Shift%% + %%Z%%
- **保存**：%%Ctrl%% + %%S%%
- **查找**：%%Ctrl%% + %%F%%
- **新窗口**：%%Ctrl%% + %%N%%
- **开发者工具**：%%F12%% / %%Ctrl%% + %%Shift%% + %%I%%
- **刷新**：%%F5%% / %%Ctrl%% + %%R%%

## 用途

- 标注快捷键与组合键
- 编写操作指引、命令行说明
- 工具栏按钮名称展示

::: warning 仅支持纯文本
`%%` 之间的内容作为纯文本渲染，不会解析 Markdown 语法。若需更复杂的样式，请配合 [属性语法](./attrs.md) 给 `<span>` 添加 class。
:::
